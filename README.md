### Featuring

- React - (UI Framework)
- Next - (Routing, Build tool uses Webpack)
- TypeScript - (Static Types)
- Webpack - (Module Bundler)

## How To Run

Running the app in dev mode:

`npm install`

`npm start`

To run in production mode:

`npm run build && npm run start`
